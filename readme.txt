Для запуска утилиты нужно:

	1) Дать права доступа к папке и всем ее файлам 777
	2) Запустить в браузере 

!!Важно:
	1) Утилита в тестовом режиме настроена на запись 20 ТТ которые уже записаны.
	   Для того чтобы изменить это в файле functions.php в 61 строке изменить число 20 на любое другое не большее 620

	2) Утилита работает через тестовый сервер. 
	   Чтоб изменить - файл functions.php строка 144 и 370 заменить url на необходимый.

Как пользоваться:

	1) Заполнить все поля и нажать на Create.(если нет api key можно использовать этот AIzaSyCfzixXn3aD85-br3_ec18CFUxRvl9oAjo)
	2) Если необходимо очистить поля для нового запроса нажать кнопку New.
	3) После запуска должен появиться загрузчик.
	4) По окончании работы утилиты сервер вернет статистику о работе.
	5) Сформированный файл отчета будет находиться в /files/report.csv
	6) В файле отчета поле json_add соответствует результату запроса на добавление ТТ, а json_get на получение данных по ТТ.
	7) В файле отчета поле validation отвечает за то прошел ли адрес валидацию через GOOGLE API.
